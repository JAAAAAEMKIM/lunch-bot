# lunch-bot

### To install dependencies:

```bash
pnpm install
```

### To build:

```bash
pnpm build
```

### To run: (build before run)

```bash
pnpm start {args}
```

### To download menus:

```bash
pnpm download
```

### args:

--dev: send to lunch-bot-dev
--dinner: send dinner
--chan2: send to group chat. (default: FE chat)
